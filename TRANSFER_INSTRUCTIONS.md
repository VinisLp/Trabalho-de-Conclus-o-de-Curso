# Instruções para Transferência do Repositório

## Após criar a organização "UrbApp-tcc" no GitHub:

### 1. Transfira o repositório no GitHub:
- Vá para: https://github.com/VinisLp/Trabalho-de-Conclus-o-de-Curso
- Settings → Transfer ownership
- Selecione "UrbApp-tcc" como novo proprietário
- Confirme a transferência

### 2. Atualize o remote local:
```bash
git remote set-url origin https://github.com/UrbApp-tcc/Trabalho-de-Conclus-o-de-Curso.git
```

### 3. Teste o push:
```bash
git push -u origin main
```

### 4. Limpe este arquivo:
```bash
rm TRANSFER_INSTRUCTIONS.md
```

## URLs importantes:
- Organização: https://github.com/UrbApp-tcc
- Repositório: https://github.com/UrbApp-tcc/Trabalho-de-Conclus-o-de-Curso
