# Guia de Contribuição

Obrigado por contribuir para o UrbApp! Para manter o repositório organizado, siga as orientações abaixo.

## Como contribuir

1. Faça um fork do repositório.
2. Crie uma branch descritiva: `feat/nome-da-feature` ou `fix/descrição-do-bug`.
3. Sempre escreva mensagens de commit claras e use o padrão [Conventional Commits](https://www.conventionalcommits.org/).
4. Atualize ou adicione testes quando necessário.
5. Submeta um pull request usando o template disponível.

## Padrões de código

- JavaScript/TypeScript deve usar ESLint com regras de Airbnb.
- Estilo de código segue Prettier.

## Revisão de PR

- Forneça descrição e contexto no PR.
- Peça revisão de pelo menos um colega.
- Esteja preparado para atender a comentários e fazer ajustes.
