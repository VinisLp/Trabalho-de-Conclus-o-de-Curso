# Roadmap do UrbApp

## Fases do projeto

### MVP
- Cadastro/login de usuário
- Envio de report com foto e localização
- Visualização de reports em mapa
- Sistema de pontuação básico

### Pós-MVP
- Níveis e insígnias
- Leaderboard público
- Filtro por categorias de report
- Notificações push

### Lançamento beta
- Testes com usuários reais
- Correções de bugs e ajustes de UX

### Versão 1.0
- Integração com órgãos municipais
- Estatísticas de uso e relatórios
