# UrbApp

Aplicativo gamificado de monitoramento de reportes urbanos.

## 📌 Visão Geral

**UrbApp** permite que cidadãos registrem ocorrências urbanas (buracos, iluminação, lixo, etc.) e sejam recompensados por meio de um sistema de gamificação.

### 🎯 Objetivos do projeto

- Fornecer uma interface móvel intuitiva para envio de reports.
- Visualizar ocorrências em mapa com geolocalização.
- Estimular o engajamento dos usuários com pontos, níveis e recompensas.
- Integrar com APIs públicas de cartografia e geolocalização.
