# 🚀 Guia: Migração para Organização UrbApp-tcc

## 📋 Checklist - Antes de Transferir

### 1. Criar Organização no GitHub
- Vá para: https://github.com/organizations/new
- **Organization name**: `UrbApp-tcc`
- **Contact email**: Seu email
- **Organization type**: Education (para TCC)
- **Plano**: Free (gratuito)

### 2. Configurar Organização
Após criar:
- Vá para Settings da organização
- **Member privileges**: Definir permissões dos membros
- **Repository permissions**: Configurar acesso aos repositórios

### 3. Transferir Repositório
- Vá para: https://github.com/VinisLp/Trabalho-de-Conclus-o-de-Curso
- Settings → Transfer ownership
- **New owner's GitHub username or organization name**: `UrbApp-tcc`
- Confirme a transferência

### 4. Convidar Colaboradores
Após transferência:
- Vá para Settings do repositório na organização
- **Collaborators and teams** → Add people
- Convide outros membros da equipe
